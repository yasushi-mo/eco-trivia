# エコ・トリビア アプリ 外部設計書

## 1. 画面構成と動作

アプリは極めてシンプルな**単一画面（One-Screen UI）**で構成され、画面遷移は行いません。

### メイン画面（トリビアビューア）

| 要素名                       | 配置場所         | 目的・動作                                                                                                             |
| :--------------------------- | :--------------- | :--------------------------------------------------------------------------------------------------------------------- |
| **トリビア表示エリア**       | 画面中央         | ランダムに取得されたエコ・トリビアのテキストを大きく表示する。                                                         |
| **トリビアテキスト**         | エリア内         | 100文字以内（目安）の簡潔な解説文。                                                                                    |
| **「次のトリビアへ」ボタン** | 画面下部中央     | タップするとデータベースから**新しいトリビアをランダムに取得**し、画面中央のトリビア表示エリアの内容を即座に更新する。 |
| **ロードインジケーター**     | 画面中央（任意） | トリビア取得中の待ち時間（1秒未満）が発生した場合にのみ表示する。                                                      |

### 画面遷移

画面遷移は行わず、「次のトリビアへ」ボタンのタップにより、画面内のコンテンツ（トリビア表示エリア）を動的に更新します。

## 2. UI デザインのコンセプト

### 配色

| 項目                 | カラーパレット                                             | 説明                                                         |
| :------------------- | :--------------------------------------------------------- | :----------------------------------------------------------- |
| **背景色**           | 白（`#FFFFFF`）または非常に薄い灰色（`#F8F8F8`）           | 清潔感とテキストの可読性を最優先する。                       |
| **アクセントカラー** | 自然な緑（例：`#4CAF50`）または爽やかな青（例：`#2196F3`） | メインカラーとして、環境・エコをイメージさせる色を使用する。 |
| **テキストカラー**   | 濃い灰色（`#333333`）                                      | 黒に近い色で、背景に対するコントラストを確保する。           |
| **ボタンカラー**     | アクセントカラーと白の組み合わせ                           | ボタンは視認性が高く、タップしやすいデザインとする。         |

### レイアウト

- **レスポンシブデザイン**: モバイルファーストとし、画面サイズに応じてフォントサイズやボタンのサイズが適切に調整されるようにする。
- **中央配置**: 主要なコンテンツ（トリビアテキスト）は画面の中央に配置し、視線の誘導を容易にする。
- **余白**: 十分なパディング（余白）を設け、要素同士が密集しない清潔な印象を与える。

### フォント

- **種類**: 読みやすく、親しみやすい**ゴシック体**を使用する。
- **トリビアテキスト**: 他の要素よりも**大きく、太字**で表示し、情報の核であることを強調する。

## 3. ユーザー体験（UX）

- **シンプルさ**: ユーザー登録やログインは一切不要。アプリを開けばすぐにコンテンツ（最初のトリビア）が表示される。
- **直感的な操作**: ユーザーアクションは「次のトリビアへ」ボタンのタップのみに限定し、迷うことなく操作できる。
- **即時性**: ボタンタップから新しいトリビアが表示されるまでの時間（パフォーマンス）を極限まで短縮し、サクサクとした心地よい閲覧体験を提供する。
