# エコ・トリビア アプリ 要件定義書

## 1. 機能要件

- **トリビア表示機能**: アプリ起動時、または画面上のボタンをタップするたびに、データベースからランダムにエコに関するトリビア（解説文）を 1 件取得し、表示します。
- **コンテンツ形式**: 表示されるコンテンツは、純粋な事実・解説文（トリビア）のみとします。
- **コンテンツ表現**:
  - 解説文は簡潔に（100 文字以内を目標に）まとめます。
  - 具体的な数字や比較を用いることで、内容の面白さと理解を促します。
- **画面構成**:
  - トリビアの文章を大きく中央に表示する、極めてシンプルな 1 画面構成とします。
  - 画面下部には、新しいトリビアを表示するための**「次のトリビアへ」ボタン**のみを配置します。
- **コンテンツ量**: 初期リリース時点で、少なくとも 100 件以上のトリビアデータを準備します。

## 2. 非機能要件

- **パフォーマンス**: クイズの読み込みや画面遷移は、1 秒以内に行うこと。
- **デザイン**: 青や緑を基調とした、清潔感のあるデザインとする。
- **互換性**: 主要なモダンブラウザ（Google Chrome、Safari、Firefox）で正常に動作すること。
- **ユーザー体験**: ユーザー登録不要で、アプリを開くだけでクイズに挑戦できるようにする。
